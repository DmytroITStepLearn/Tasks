Завдання

Створити абстрактний клас Employee із методом ToPay, який повертає int.

Забезпечити, щоб класи Student, Manager, і Worker наслідували клас Employee.

Інтерфейс IHasSalary:

Створити інтерфейс IHasSalary у папці Models.
Додати до нього властивість int Salary та метод void SetBonus.
Реалізувати інтерфейс у класах Manager і Worker.

Властивості класів:
Для класу Student створити властивість double AvarageGrade.
Для класу Worker створити властивість int WorkedHours.
Для класу Manager створити властивість int EmployeeCount.

Методи ToPay:
Для класу Student реалізувати метод ToPay, який повертає 2000, якщо середня оцінка = 4 після 
стандартного заокруглення (до вищого).

Для класу Worker реалізувати метод ToPay, який повертає зарплатню помножену на пропрацьовані години, 
або зарплатню збільшену на 50 та помножену на пропрацьовані години, якщо пропрацьовані години > 40.

Для класу Manager реалізувати метод ToPay із логікою:
Якщо кількість робітників < 5, повернути зарплатню.
Якщо EmployeeCount <= 10 і >= 5, повернути зарплатню та додати 100 за кожного робітника менеджера.
Якщо EmployeeCount > 10, повернути Salary вдвічі більшу.

Метод SetBonus:
У класі Worker метод SetBonus додає значення бонусу до зарплатні.
У класі Manager метод SetBonus додає значення бонусу до зарплатні і додає 10 за кожного робітника менеджера.
